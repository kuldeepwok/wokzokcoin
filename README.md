# WokzokCoin (WKC) - Excel-Based Token System

A hybrid Excel-based digital asset system designed to emulate blockchain-like immutability, transparency, and traceability without complex infrastructure.

## Features

- **Excel-Based Ledger**: Uses Microsoft Excel as a decentralized-like ledger container
- **Hash Chaining**: Implements SHA256 hash chaining for immutable transaction history
- **QR Verification**: Each token has a unique QR code for verification
- **Multi-Page Interface**: Home, Wallet, Transactions, Excel Ledger, and Whitepaper pages

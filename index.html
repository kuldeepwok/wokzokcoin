Executive Summary

WokzokCoin (WKC) is a hybrid Excel-based digital asset system designed to emulate blockchain-like immutability, transparency, and traceability â€” without relying on SQL databases or external blockchain networks.

Problem Statement

Most blockchain systems are costly and infrastructure-heavy. WokzokCoin provides a no-cost, Excel-based alternative ideal for small-scale ecosystems, education, and transparent tokenized experiments.

Objectives

  • Create a tamper-evident, Excel-based ledger system for token tracking.
  • Provide blockchain-like immutability through hash chaining.
  • Allow transparent and verifiable token issuance without servers.
  • Enable learning and experimentation with blockchain logic in Excel.

System Architecture

The WKC system uses an Excel workbook as a decentralized-like ledger container. Each sheet represents a node, and each row represents a transaction block. The VBA macro engine drives the blockchain-like functionality, computing SHA256 hashes and linking each block to its predecessor.
